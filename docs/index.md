Welcome to the `python-git-template` documentation. This repository is a template
for quickly bootstrapping a PyPI-ready Python package complete with CI/CD via
GitHub Actions, Docker support, and Sphinx documentation hosted on ReadTheDocs.

## Getting Started

Replace all occurrences of `pkg-name` with your package name, fill in the
placeholders in `pyproject.toml`, update the `README.md`, and start building.
See [Installation](./about/installation.md) for step-by-step instructions.

## Motivation & Design

Maintaining a well-structured Python package from the start saves significant
time. This template provides sensible defaults for packaging, linting, testing,
and documentation so you can focus on writing your package code.
