# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Instead, email the maintainer directly at **dpramanik.official@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce it
- Potential impact assessment
- Any suggested fixes (optional)

You will receive a response within 48 hours. Once the vulnerability is confirmed and
patched, a public disclosure will be made.
